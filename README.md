# Multi-Agent AI Customer Support Assistant

This project is a web-based AI-powered customer support assistant using Multi-Agent Systems, RAG, Vector Database, and LLM integration.

## Features

- User Authentication
- AI Chat Interface
- Intent Detection
- Agent Routing
- Billing Agent
- Technical Support Agent
- Product Agent
- Complaint Agent
- FAQ Agent
- RAG-based Knowledge Retrieval
- Conversation History
- Escalation for unresolved issues

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Python FastAPI
- MongoDB
- FAISS
- Sentence Transformers
- Gemini API

### Deployment
- Vercel for Frontend
- Render/Railway for Backend
- MongoDB Atlas for Database
